# Sales Playbook

An [Agent Skill](https://skills.sh) that gives AI coding agents expert-level sales knowledge for cold outreach, objection handling, closing techniques, follow-up sequences, and persuasion psychology.

Built for B2B SaaS, digital products, and service businesses.

## What's Inside

| Reference | Content |
|-----------|---------|
| [Cold Outreach](sales-playbook/references/cold-outreach.md) | Email (PAS, AIDA, BAB), WhatsApp templates, cold call scripts |
| [Objection Handling](sales-playbook/references/objections.md) | Responses for 7 common objections with multiple techniques each |
| [Closing & Follow-Up](sales-playbook/references/closing-and-followup.md) | 10+ closing techniques and a 30-day multi-channel follow-up sequence |
| [Methodologies](sales-playbook/references/methodologies.md) | SPIN, Challenger, Sandler, Straight Line, Gap Selling |
| [Persuasion](sales-playbook/references/persuasion.md) | Cialdini's 7 principles, Chris Voss negotiation, storytelling frameworks |
| [Mindset](sales-playbook/references/mindset.md) | Seller psychology, state management, 10X rule |

## Installation

```bash
npx skills add agusgarcia3007/sales-playbook
```
